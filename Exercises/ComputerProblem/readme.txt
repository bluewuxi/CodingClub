Computer Problem

Statement
You have n computers to turn on. In 1 operation, you can turn on any of the n computers. And when the i - 1 th and the i + 1 th computer are both turned on, the ith computer is turned on.
Your task is to find how many ways to turn on the n computers. Output your answer modulo by M.

Input
2 space-separated integers n and M.

Output
How many ways to turn on the n computers modulo by M.

Example input 1
3 100000007

Example output 1
6

Example input 2
4 100000007

Example output 2
20

Example input 3
400 234567899

Example output 3
20914007

Constraints
1 <= n <= 400;
108 <= M <= 109
